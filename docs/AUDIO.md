# 音频文件说明

本项目的音频文件存储在云服务器上，未包含在 Git 仓库中。

## 文件结构

音频文件应放置在以下目录：

```
data/uploads/
└── {语言对编码}/           # 如 de-en, zh-en
    ├── source/             # 原始音频
    ├── engine1/            # 引擎1翻译音频
    └── engine2/            # 引擎2翻译音频
```

## 获取音频文件

请联系项目管理员获取音频文件下载链接。

## 手动部署

1. 下载音频文件压缩包
2. 解压到 `data/uploads/` 目录
3. 在管理后台点击"扫描音频"按钮，系统会自动匹配语料ID

## 文件命名规范

音频文件名应与语料ID匹配：

- 文件名：`{语料ID}.wav`
- 示例：`de-en-DE-06-Jona_Airport_Security-check_014.wav`
